REQUIREMENTS:
- MinGW (g++ compiler)
- Windows 7/10/11
- Internet
INSTALLING MinGW:
1. Download from https://winlibs.com/ or through MSYS2
2. Add bin/ folder to PATH
RUNNING:
Double-click build.bat
USAGE:
1. Obtain Bot Token from @BotFather in Telegram
2. Launch OPIUM.exe
3. Enter the token
4. Select menu:
  - Listen: receives all incoming messages
  - Send: sends a message to the Chat ID
GETTING Chat ID:
- Type something to the bot
- In "Listen" mode, the ID will appear in brackets
